Logic Explanation

My app uses a lootArray to store each item as an object with name and value properties. When items are added, I use a for loop to calculate the running total and another for loop to render each item to the DOM. Conditional if statements validate that party size ≥ 1, loot names aren't empty, and values are valid positive numbers. When splitting, I divide the total by party size and format with toFixed(2).
